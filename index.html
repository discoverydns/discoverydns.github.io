DiscoveryDNS

Introduction

This is the home page of the DiscoveryDNS solution, which includes the following projects:

  • dnsapi-client, a Java client to the DiscoveryDNS Server REST API.
  • ddns-dnsjava, a fork of the jdnssec-dnsjava project, which was originally a "minor fork of DNSjava, to support jdnssec-tools".
